# $addTimestamp

Добавить в подвал эмбеда время вызова команды.

### Использование
 
```php
$addTimestamp[индекс]
```

### Опции


| Опция | Описание | Тип | Обязательно? |
|--------|-------------|------|----------|
| индекс | номер эмбеда | число | да |


## Пример:

```javascript
bot.command({
  name: 'embed-timestamp',
  code: `
  $title[1; Команда была вызвана] 
  $addTimestamp[1]
  `
});
```
