

# $addField

Добавить поле для эмбеда, это как свой мини-эмбед в большом эмбеде')

### Использование
 
```php
$addField[индекс;заголовок;текст;отдельно]
```

### Опции


| Опция | Описание | Тип | Обязательно? |
|--------|-------------|------|----------|
| индекс | номер эмбеда (максимум 64) | число | да |
| заголовок |

bot.command({
