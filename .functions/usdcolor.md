# $color

Добавить цвет для эмбеда.

### Использование
 
```php
$color[индекс;цвет]
```

### Опции


| Опция | Описание | Тип | Обязательно? |
|--------|-------------|------|----------|
| индекс | номер эмбеда | число | да |
| цвет | HEX code | текст | да |


## Пример:

```javascript
bot.command({
  name: 'embed-color',
  code: `
  $title[1;Цвет]
  $color[1;00FF00]
  `
});
```
