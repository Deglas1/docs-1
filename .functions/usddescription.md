# $description

.

### Использование
 
```php
$description[индекс;текст]
```

### Опции


| Опция | Описание | Тип | Обязательно? |
|--------|-------------|------|----------|
| индекс | номер эмбеда (максимум 64) | число | да |
| текст | описание эмбеда (максимум 4096 символов) | текст | да |


## Пример:

```javascript
bot.command({
  name: 'embed-description',
  code: `
  $description[1;А что тут писать?]
  `
});
```
