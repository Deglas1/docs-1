# $function
Выполняет указаный код

## Опции для этой функции
| Опция |  Описание | Тип опции | Обязательно ли указывать | 
| ------ | ----------- | --------- | ------------------------ |
| Код | то, что нужно выполнить | Текст | Да |

## Пример использования
```javascript
bot.command({
name: "function",
code: `$function`
})
```
