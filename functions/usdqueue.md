
# $queue

Создаёт список очереди всех треков используя {} функции
### Использование
 
```php
$queue[]
```

### Опции


| Опция | Описание | Тип | Обязательно? |
|--------|-------------|------|----------|
| param 1 | Это показывает `param 1` | ТЕКСТ | да |
| param 2 | Это показывает `param 2` | ЧИСЛО | нет |


## Пример(ы)

```javascript
bot.command({
  name: 'function-name',
  code: `
  $function[index;aoijs]
  `
// Возвращает: ...
});
```
