# $addReactions
Добавляет реакции к сообщению, которое создал бот
### Использование
```php
$addReactions[...реакции]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| ...реакции | реакции, которые нужно поставить на сообщение | перечисление | Да |  
## Пример(ы)

```javascript
bot.command({
  name: '$addReactions',
  code: `
$addReactions[:joy:;:white_check_mark:]`
})
```
