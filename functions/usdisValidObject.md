
# $isValidObject 

Проверяет, правильно ли сделан объект

### Использование
 
```php
$isValidObject[объект]
```

### Опции


| Опция | Описание | Тип | Обязательно? |
|--------|-------------|------|----------|
| объект | объект, который нужно проверить | ОБЪЕКТ | да |


## Пример(ы)

```javascript
bot.command({
  name: 'is-valid-object',
  code: `
  $isValidObject[{ "объект": { "название": "свойство" }]
  `
// Возвращает: true
});
```
