# $addFields
Добавляет несколько полей для встроенного сообщения
### Использование
```php
$addFields[название:текст:отдельное поле?;поля]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| поля |  |  | Да |  
## Пример(ы)

```javascript
bot.command({
  name: '$addFields',
  code: `
$addFields[hello:world:no;how:are you?:no]`
})
```
