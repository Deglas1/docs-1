# $banCount
Возвращает количество банов на указанном сервере.
### Использование
```php
$banCount[сервер?]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| сервер | сервер, на котором нужно узнать количество банов | айди | Нет |  
## Пример(ы)

```javascript
bot.command({
  name: '$banCount',
  code: `
$banCount[$guildid]`
// Возвращает: 57
})
```
