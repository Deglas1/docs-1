# $botCount
Возвращает количество ботов на указанном сервере.
### Использование
```php
$botCount[сервер?]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| сервер | сервер, на котором нужно получить количество ботов | айди | Нет |  
## Пример(ы)

```javascript
bot.command({
  name: '$botCount',
  code: `
$botCount[$guildid]`
// Возвращает: 21
})
```
