# $dm
Эта функция отправляет текст в личные сообщения пользователя через ID
### Использование
```php
$dm[айди?]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| айди | Айди пользователя, которому мы отправляем текст | число | Нет |  
## Пример(ы)

```javascript
bot.command({
  name: 'dm',
  code: `
$dm[973803066266648617] Привет!`

})
```
