# $botOwnerID
Возвращает айди владельца (-ов) бота на портале разработчиков
### Использование
```php
$botOwnerID[разделитель?]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| разделитель | разделитель для айди | текст | Нет | 
## Пример(ы)

```javascript
bot.command({
  name: '$botOwnerID',
  code: `
$botOwnerID[ , ]`
// Возвращает: 495678595650060 , 967004504532290
})
```
